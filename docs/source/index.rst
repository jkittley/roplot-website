Welcome to Roplot's documentation!
==================================
This projects aim is develop a set of tools and instruction so that anyone can build and develop robic drawing machines which rotate around a central pivot. This project is a work in progress and we hope to soon develop the documentation further.

Features
--------

- Custom printing language
- Novel hardware
- Easy customisation
- Well documented
